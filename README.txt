Requisitos

O site tem como objetivo oferecer o serviço de conversão e download de vídeos, isto é,
o cliente pode copiar um link de um vídeo em alguma rede social, como facebook e instagram, cola o link no 
site, e nosso sistema vai converter e disponibilizar um download para o cliente baixar este vídeo.

Funcionais

    1- O site deve ter um input para inserção do link do vídeo.
    2- O site deve ter um botão para conversão.
    3- O site deve ter um botão para download.
    4- O site deve ter um botão para voltar.
    5- O site deve ter um botão para converter outro vídeo que vai ter a mesma função de voltar.

Não Funcionais

    6- O site deve ter uma interface amigiável;
    7- O site deve realizar o serviço rápido;
    8- O site não deve ter propagandas;
    9- O site deve ser minimalista;
    10- O site deve ter tratamento de erros;
